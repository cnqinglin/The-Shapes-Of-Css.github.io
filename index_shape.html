the-shapes-of-css

box-sizing:border-box; // 已设定的高度和宽度之内添加内边距和边框
box-sizing:content-box; // 已设定的高度和宽度之外添加内边距和边框

圆：用到的属性：border-radius :100%;
例子：
代码如下：

width: 100px;
height: 100px;
background-color: skyblue;
border-radius: 100%;

椭圆：用到的属性：border-radius :100%/50%;
例子：
代码如下：

width: 100px;
height: 100px;
background-color: skyblue;
border-radius: 100%/50%;

上三角形：用到的属性：border:1px solid transparent;
三角形有上、下、左、右三角形，原理一样；
例子：
代码如下：

width: 0px;
height: 0px;
border-bottom:10px solid skyblue;
border-right:10px solid transparent;
border-left:10px solid transparent;
